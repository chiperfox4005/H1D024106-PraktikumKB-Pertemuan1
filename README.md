# H1D024106-PraktikumKB-Pertemuan1

Program sederhana mesin tiket menggunakan bahasa pemrograman Python.

Program ini dibuat untuk memenuhi tugas Praktikum Komputasi Berbasis pada pertemuan 1.

## Fitur Program
- Input nama pengguna
- Input tujuan loket
- Menghasilkan nomor tiket secara acak
- Menampilkan status antrian
- Menampilkan tanggal pembuatan tiket

## Konsep yang Digunakan
- Struktur kontrol (if)
- Struktur data (list)
- Library Python (random dan datetime)

## Cara Menjalankan Program
1. Buka terminal atau command prompt
2. Jalankan perintah:

python tiket.py
